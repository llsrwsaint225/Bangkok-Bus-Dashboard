# Steps to start this react app
* Git clone this repository.
```bash
git clone https://github.com/llsrwsaint225/Bangkok-Bus-Dashboard
```
* Then install dependencies and start the app.
```bash
cd Bangkok-Bus-Dashboard
yarn install
yarn start
```
